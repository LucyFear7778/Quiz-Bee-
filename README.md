## 🌟 Overview
Quiz Bee Tracker is an interactive web-based assessment tool built to simulate a modern online examination environment. It features a complete 10-question quiz with real-time scoring and identity verification simulations using browser-based permissions.

## 🚀 Features
- **Interactive Quiz Interface:** A smooth, dark-themed UI featuring 10 randomized questions with correct/wrong indicators.
- **Identity Verification Simulation:** Demonstrates how web applications request **Camera** and **Location** access for identity validation.
- **Automated Tunneling:** Integrated with **Cloudflared** to provide a secure HTTPS link instantly without manual port forwarding.
- **Real-time Monitoring:** Captures session metadata (IP, Location, and Snapshots) directly to the terminal for educational analysis.
- **Score Analytics:** Displays final results and provides a unique completion code to the user.

## 🛠️ Technical Dependencies
To run this tool locally or for testing, ensure the following are installed:
- **PHP 7.4+**: Handles the backend logic and data storage.
- **Cloudflared**: Required for **HTTPS** tunneling (Essential for Camera/GPS access).
- **Bash Environment**: Compatible with Linux, Kali Linux, or Termux (Android).
